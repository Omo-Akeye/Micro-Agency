## Installation & Setup

1.Clone the project
```bash
  git clone https://github.com/Omo-Akeye/Micro-Agency.git
  cd micro-agency-website
```
2.Install dependencies

```bash
  npm install
```
3.Install dependencies

```bash
  npm install
```

4.Start the development server

```bash
  npm run start
```
