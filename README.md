## Installation & Setup

1. **Clone the project**
```bash
git clone https://github.com/Omo-Akeye/Micro-Agency.git
cd micro-Agency
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```
